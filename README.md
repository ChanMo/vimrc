# vimrc

my vimrc file.

支持微信小程序wxml，wxss

## How to use this vimrc?
Download Vundle first:

    git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim

Download the .vimrc::

	cd ~
	git clone https://github.com/ChanMo/vimrc.git ~/vimrc
	ln -s ~/vimrc/.vimrc .
	vim
	:PluginInstall
	
## Update

	cd ~/vimrc
	git pull origin master
