# vimrc

my vimrc file.

## How to use this vimrc?
Just do this:

	cd ~
	git clone https://github.com/ChanMo/vimrc.git ~/vimrc
	ln -s ~/vimrc/.vimrc .
	vim
	:PluginInstall
	
## Update

	cd ~/vimrc
	git pull origin master
